# 飞机大战游戏

基于C++开发的经典飞机大战游戏，运用多种设计模式实现。

## 游戏特性

- 玩家飞机控制与射击
- 3种类型敌机（大型、中型、小型）
- 碰撞检测与爆炸效果
- 分数统计

## 技术栈

- **语言**：C++
- **IDE**：Visual Studio
- **图形库**：EasyX

## 设计模式应用

| 模式 | 应用场景 |
|------|---------|
| **工厂模式** | FacFoe工厂类管理3种敌机的创建，便于扩展新类型 |
| **观察者模式** | 游戏事件通知机制（敌机被击毁、分数更新等） |

## 核心实现

### 敌机继承体系

```
       Foe (基类)
        │
   ┌────┼────┐
   │    │    │
FoeBig FoeMid FoeSma
(大型) (中型) (小型)
```

不同敌机具有差异化属性：
- 大型敌机：血量高、速度慢、分数高
- 中型敌机：血量中、速度中、分数中
- 小型敌机：血量低、速度快、分数低

### 碰撞检测

基于矩形包围盒（AABB）碰撞检测，检测：
- 玩家子弹与敌机碰撞
- 敌机子弹与玩家碰撞
