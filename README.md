# 飞机大战游戏

基于C++开发的经典飞机大战游戏，运用多种设计模式实现。

## 游戏特性

- 玩家飞机控制与射击
- 多种类型敌机自动生成
- 道具系统（血量恢复、武器升级等）
- 碰撞检测与爆炸效果
- 分数统计与关卡难度递增

## 技术栈

- **语言**：C++
- **IDE**：Visual Studio
- **图形库**：Windows GDI

## 设计模式应用

| 模式 | 应用场景 |
|------|---------|
| **工厂模式** | 管理不同类型敌机和道具的创建，便于扩展新类型 |
| **观察者模式** | 游戏事件通知机制（敌机被击毁、道具被拾取等） |
| **单例模式** | 游戏全局状态管理、资源配置管理 |

## 项目结构

```
cpp-plane-battle/
├── GameMain.cpp          # 游戏入口
├── GameManager.h/cpp     # 游戏管理器（单例）
├── Plane.h/cpp           # 飞机基类
├── EnemyPlane.h/cpp      # 敌机类
├── Bullet.h/cpp          # 子弹类
├── Item.h/cpp            # 道具类
├── Factory.h/cpp         # 工厂类
└── Observer.h/cpp        # 观察者模式实现
```

## 运行方式

1. 使用Visual Studio打开 `飞机大战开发.sln`
2. 编译运行即可
